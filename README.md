## 💡 What is this?

**fahh** is a VS Code extension designed for those who need a little extra "feedback" when things go wrong. Every time you run a command that fails or make a mistake in your terminal, the extension plays the iconic "fahh" sound. 

It's the ultimate motivational tool—if motivation was replaced by mild verbal disappointment.

---

## 🚀 Features

* **Instant Regret:** Automatically triggers the "fahh" sound effect on terminal command errors.
* **Seamless Integration:** Works quietly in the background until you mess up.
* **Lightweight:** Won't lag your environment, even if you're having a really bad coding day.

## 🛠️ Installation & Requirements

1. Install the extension from the VS Code Marketplace.
2. Ensure your system volume is up (but maybe keep your headphones on if you're in a library).
3. **Dependencies:** No external drivers required!

## ⚙️ Extension Settings

Tailor your experience via the following settings:

* `fahh.enabled`: Toggle the sound on or off.
* `fahh.volume`: Adjust the level of disappointment (0.1 to 1.0).

## 🚩 Known Issues

* Might cause emotional damage if you're struggling with a complex regex.
* Currently only triggers on standard exit codes; custom shell configurations may vary.
